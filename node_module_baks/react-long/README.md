# react-long
Warning: works only on devices that support touch.

# Demo

[http://react-long.surge.sh](http://react-long.surge.sh)

# Example

```js
<div>
  <LongPress
    time={1000}
    onLongPress={() => alert('pressed for 1s')}
    onPress={() => alert('pressed')}
  >
    <div>
      Hello world
    </div>
  </LongPress>
</div>
```
